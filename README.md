# 🎮 Playnite Extensions - Soluções de Valor para Gamers

Bem-vindo ao meu repositório de extensões para o [Playnite](https://playnite.link/).

## 🚀 Sobre o Projeto e Minha Visão
Diferente da maioria dos repositórios de código, este projeto não foi criado por um desenvolvedor profissional de software. **Eu sou um Product Manager (PM)**. 

No meu dia a dia, trabalho criando produtos que entregam valor real aos usuários. Como um entusiasta de games e usuário do Playnite, percebi lacunas que afetavam minha experiência. Decidi então "colocar as mãos na massa" e desenvolver soluções que resolvessem os meus problemas e, potencialmente, os de outras pessoas que compartilham da mesma paixão.

Este repositório é fruto dessa jornada: criar ferramentas úteis, simples e que melhorem a forma como interagimos com nossas bibliotecas de jogos.

> [!IMPORTANT]
> Como não sou desenvolvedor por formação, meu foco está na funcionalidade e no valor entregue. Se você encontrar bugs ou tiver sugestões de melhoria no código ou no produto, **todo feedback é extremamente bem-vindo!** Este é um espaço aberto para contribuições.

## 🛠️ Extensões Disponíveis

Este repositório contém as seguintes extensões (e mais estão por vir!):

### 1. 🔍 [Busca de Jogos Locais](./BuscaDeJogosLocais)
Uma ferramenta robusta para quem prefere gerenciar seus jogos manualmente sem depender de launchers externos.
- **Destaque**: Identifica automaticamente em qual HD o jogo está instalado, permitindo filtros físicos na sua biblioteca.
- **Status**: Pronta para uso.

### 2. 📅 [Retrospective Steam](./RetrospectiveSteam)
Uma experiência visual completa para relembrar sua jornada gamer ao longo do ano.
- **Destaque**: Inspirada no "Steam Replay", com linha do tempo em zigzag, gráficos de hábitos e personas personalizadas.
- **Status**: Pronta para uso.

### 3. ✨ [Futura Extensão] (Em Breve)
Já estou trabalhando em uma terceira solução para o ecossistema Playnite. Fique atento às atualizações!
