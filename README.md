# 🎮 Playnite Extensions - Value-Driven Solutions for Gamers

Welcome to my repository of extensions for [Playnite](https://playnite.link/).

## 🚀 About the Project and My Vision
Unlike most code repositories, this project was not created by a professional software developer. **I am a Product Manager (PM)**.

In my day-to-day life, I focus on building products that deliver real value to users. As a gaming enthusiast and Playnite user, I noticed gaps that affected my experience. I decided to get my hands dirty and develop solutions that solved my own problems and, potentially, those of others who share the same passion.

This repository is the result of that journey: creating useful, simple tools that improve how we interact with our game libraries.

> [!IMPORTANT]
> Since I am not a developer by training, my focus is on functionality and delivered value. If you find bugs or have suggestions for improvements in the code or the product, **all feedback is extremely welcome!** This is an open space for contributions.

---

## 🛠️ Available Extensions

This repository contains the following extensions (with more to come!):

### 1. 🔍 [Local Games Search](./BuscaDeJogosLocais)
A robust tool for those who prefer to manage their games manually without relying on external launchers.
- **Highlight**: Automatically identifies which HD the game is installed on, allowing physical filters in your library.
- **Status**: Ready for use.

### 2. 📅 [Retrospective Steam](./RetrospectiveSteam)
A complete visual experience to relive your gaming journey throughout the year.
- **Highlight**: Inspired by "Steam Replay," featuring a zigzag timeline, habit charts, and personalized personas.
- **Status**: Ready for use.

### 3. ✨ [Future Extension] (Coming Soon)
I am already working on a third solution for the Playnite ecosystem. Stay tuned for updates!
